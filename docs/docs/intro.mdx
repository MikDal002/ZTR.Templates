# Getting Started

Ta strona powinna zawierac informację na temat projektu, na którym aktualnie się znajduje.

Pozostałe strony w kategoriach "Tutorial - *" pokazują jak korzystać z tego silnika do tworzenia dokumentacji.
Dlatego też tutaj zostały.

## Uruchamianie strony

Run the development server:

```bash
cd my-website
npm run start
```

The `cd` command changes the directory you're working with. In order to work with your newly created Docusaurus site, you'll need to navigate the terminal there.

The `npm run start` command builds your website locally and serves it through a development server, ready for you to view at http://localhost:3000/.

Open `docs/intro.md` (this page) and edit some lines: the site **reloads automatically** and displays your changes.

## Wyszukiwarka

Uwaga, wyszukiwarka nie działa po uruchomieniu przez `npm start`, wymaga ona zbudowania strony przez `npm run build` i uruchomienia jej przez `npm run serve`.
